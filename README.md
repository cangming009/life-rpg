## 📖 项目简介

**生活西游** 是一款基于 HTML5 的单页 RPG 网页游戏，旨在通过**游戏化（Gamification）**的方式帮助玩家养成良好的生活习惯。

在这个游戏中，你不需要整天盯着屏幕。**现实生活才是你的主战场**：
*   控制住食欲减肥？👉 **获得经验+金币**
*   辅导孩子写完作业？👉 **获得高额奖励**
*   攒够了“自律币”？👉 **购买装备，挑战“拖延症”、“手机瘾”等心魔BOSS！**

## ✨ 核心特性

*   **📱 全端适配**：
    *   **手机端**：完美全屏适配，体验如同原生 APP。
    *   **电脑端**：居中模拟手机窗口，沉浸式体验。
*   **💾 自动存档**：利用浏览器 `localStorage` 技术，刷新网页或关闭浏览器，进度永远不会丢失。
*   **⚔️ 回合制战斗**：经典的 RPG 战斗模式，包含普通攻击、必杀技（消耗生命值）和暴击机制。
*   **🛍️ 经济循环**：通过完成现实任务赚取金币，在商店购买属性道具（如“降噪耳机”、“五年高考三年模拟”）。
*   **📈 无限成长**：关卡难度动态递增，挑战无止境。
