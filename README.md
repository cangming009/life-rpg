# 📜 游戏玩法与设计文档 (Game Design Document)

## 1. 核心理念 (Core Concept)

**"Life is an RPG."**
本游戏的核心循环是将**现实世界的痛苦投入（Input）** 转化为 **虚拟世界的数值奖励（Output）**。通过可视化的成长（等级提升、装备获取），为枯燥或困难的现实行为提供即时反馈。

## 2. 游戏系统详解

### 2.1 现实修行系统 (任务模块)
这是游戏的资源产出入口。玩家通过确认完成现实生活中的行为来触发。

| 任务名称 | 现实行为 | 游戏奖励 | 冷却时间 | 设计意图 |
| :--- | :--- | :--- | :--- | :--- |
| **🥗 坚持减肥** | 控制饮食/吃健康餐 | Exp +20 / 金币 +15 | 2秒 | 鼓励健康饮食，基础奖励。 |
| **📚 辅导作业** | 辅导孩子学习/自己学习 | Exp +30 / 金币 +25 | 2秒 | 高难度任务，给予高额奖励。 |
| **🧹 收拾家务** | 打扫卫生/整理房间 | Exp +15 / 金币 +10 | 2秒 | 碎片化任务，低门槛。 |
| **🏃 健身运动** | 跑步/举铁/有氧 | Exp +40 / 金币 +30 | 2秒 | 最高体能消耗，给予最高奖励。 |

*注：任务点击后会同时**回满玩家生命值**，寓意“休息与充能”。*

### 2.2 经济与装备系统 (商店模块)
消费系统，用于消耗玩家积累的“自律币”，转化为永久战斗属性。

*   **货币**：自律币 (Gold)
*   **属性类型**：
    *   `HP` (生命上限)：提高生存能力。
    *   `ATK` (攻击力)：提高清怪效率。

**特殊道具列表：**
1.  **静心口服液** (100G)：新手入门攻击装。
2.  **瑜伽垫** (200G)：基础生命装。
3.  **五年高考三年模拟** (500G)：中级攻击神器（知识就是力量）。
4.  **降噪耳机** (800G)：中级生命装（屏蔽孩子的吵闹）。
5.  **戒尺** (2000G)：高级攻击装。
6.  **快乐肥宅水** (5000G)：终极全属性提升（偶尔的放纵是巨大的力量）。

### 2.3 战斗系统 (闯关模块)
验证玩家成长的地方。

*   **机制**：回合制 (Turn-Based)。
*   **行动选项**：
    1.  **👊 普通攻击**：造成 `ATK * (0.9 ~ 1.1)` 的浮动伤害。
    2.  **✨ 必杀技**：消耗 `10% 当前生命值`，造成 `ATK * 2.5` 的巨额伤害。适用于拼命阶段。
*   **BOSS 机制**：
    *   如果战斗失败，BOSS 血量重置，玩家需回修行页面做任务回血。
    *   如果在战斗中生命归零，判负。

### 2.4 关卡成长 (无限流)
为了支持无限游玩，游戏采用动态难度算法。

*   **基础怪物池**：
    1.  手机诱惑怪
    2.  沙发土豆精
    3.  作业拖延魔
    4.  起床困难症
    5.  中年危机兽
    6.  完美的自己 (Base Boss)
*   **动态难度公式**：
    当关卡数 `Stage > 6` 时，怪物进入第二轮循环。
    *   **怪物名称**：原名 + `(增强版)`
    *   **怪物血量**：`基础血量 * (1 + 轮数 * 0.5)`
    *   **怪物攻击**：`基础攻击 * (1 + 轮数 * 0.5)`

## 3. 数值模型

### 3.1 玩家升级公式
玩家升级所需经验值呈指数级增长，防止后期升级过快。
*   `Next_Level_Exp = Previous_Level_Exp * 1.5`

### 3.2 属性成长
*   每次升级：`HP +20`, `ATK +5`
*   装备加成：直接叠加在基础属性上。

## 4. 交互设计 (UI/UX)

*   **布局**：采用底部 Tab 导航（修行/商店/背包/战斗），符合移动端用户直觉。
*   **反馈**：
    *   **点击任务**：按钮出现进度条动画（冷却反馈）。
    *   **战斗日志**：不同颜色的文字区分我方伤害（绿）、敌方伤害（红）和暴击/技能（黄）。
    *   **响应式**：PC 端限制最大宽度并居中，背景加深色遮罩，聚焦游戏区域。
